## 概要

ユーザーがタグを使ってカードをフィルタリング・検索し、特定の分野を集中的に学習できるようにする。複数タグの組み合わせ（AND/OR）にも対応。

## ユーザーフロー

### カード一覧でのフィルタ
1. **[トリガー]** ユーザーがカード一覧画面でフィルタアイコンをタップ
2. **[システム]** タグ選択UIを表示
3. **[ユーザー]** フィルタしたいタグを選択（複数可）
4. **[システム]** 選択されたタグに紐付くカードのみ表示
5. **[完了]** フィルタ結果が表示される

### 学習時のタグ指定
1. **[トリガー]** ユーザーが「学習開始」をタップ
2. **[システム]** 学習対象のタグ選択画面を表示
3. **[ユーザー]** 学習したいタグを選択（「全て」も選択可）
4. **[システム]** 選択タグに紐付くカードから復習対象を抽出
5. **[完了]** 学習セッションが開始される

## 入力/出力

| 種別 | 項目名 | 型 | 必須 | 説明 | 制約・バリデーション |
|-----|-------|---|-----|-----|-------------------|
| 入力 | tagIds | string[] | No | フィルタするタグID | 空の場合は全カード |
| 入力 | filterMode | 'AND' \| 'OR' | No | フィルタモード | デフォルト: OR |
| 入力 | searchText | string | No | テキスト検索 | 表面・裏面を検索 |
| 出力 | cards | Card[] | - | フィルタ結果 | タグ情報含む |
| 出力 | totalCount | number | - | 該当カード数 | - |

## ビジネスルール

| ID | ルール | 条件 | 結果 |
|----|-------|-----|-----|
| BR-F018-01 | ORフィルタ | filterMode='OR' | 選択タグのいずれかに紐付くカードを表示 |
| BR-F018-02 | ANDフィルタ | filterMode='AND' | 選択タグの全てに紐付くカードのみ表示 |
| BR-F018-03 | タグ未選択 | tagIdsが空 | 全カードを表示 |
| BR-F018-04 | タグなしカード | 「タグなし」フィルタ | タグが紐付いていないカードを表示 |
| BR-F018-05 | テキスト検索 | searchText指定時 | 表面・裏面に含まれるカードを表示 |
| BR-F018-06 | 複合フィルタ | タグ+テキスト | 両方の条件を満たすカードを表示 |

## ユースケース例

| シナリオ | 選択タグ | モード | 結果 |
|---------|---------|-------|------|
| TOEIC語彙を学習 | "TOEIC", "単語" | AND | 両方のタグがついたカードのみ |
| 英語関連を全て復習 | "英語", "TOEIC", "文法" | OR | いずれかのタグがついたカード |
| タグなしカードを整理 | (タグなし) | - | タグ未設定のカードのみ |

## エラーケース

| ID | 条件 | エラーメッセージ | 対処 |
|----|-----|----------------|-----|
| E-F018-01 | 該当カードなし | 条件に一致するカードがありません | フィルタ条件を変更 |
| E-F018-02 | 検索失敗 | 検索に失敗しました | リトライ |

## 受け入れ条件（AC）

- [ ] AC-01: タグを選択してカードをフィルタできること
- [ ] AC-02: 複数タグをOR条件でフィルタできること
- [ ] AC-03: 複数タグをAND条件でフィルタできること
- [ ] AC-04: テキスト検索でカードを絞り込めること
- [ ] AC-05: タグなしのカードをフィルタできること
- [ ] AC-06: フィルタ条件をクリアして全カードを表示できること
- [ ] AC-07: 学習開始時にタグを指定して学習範囲を限定できること

## 技術仕様

### Supabase連携
```typescript
// ORフィルタ
const { data: cards } = await supabase
  .from('cards')
  .select(`
    *,
    tags:card_tags(tag:tags(*))
  `)
  .eq('user_id', userId)
  .in('id',
    supabase
      .from('card_tags')
      .select('card_id')
      .in('tag_id', tagIds)
  )

// ANDフィルタ（全タグを持つカードのみ）
// PostgreSQL関数を使用するか、アプリ側でフィルタ

// テキスト検索
const { data: cards } = await supabase
  .from('cards')
  .select('*')
  .eq('user_id', userId)
  .or(`front.ilike.%${searchText}%,back.ilike.%${searchText}%`)

// タグなしカード
const { data: cards } = await supabase
  .from('cards')
  .select('*')
  .eq('user_id', userId)
  .not('id', 'in',
    supabase.from('card_tags').select('card_id')
  )
```
