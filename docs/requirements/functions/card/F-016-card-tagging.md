# F-016: カードへのタグ付け

| 項目 | 内容 |

## 概要

ユーザーがカードにタグを付与し、学習内容を分類・整理できるようにする。1つのカードに複数のタグを付与可能。カード作成時・編集時にタグを設定できる。

## ユーザーフロー

### カード作成時のタグ付け
1. **[ユーザー]** カード作成フォームでタグ入力欄をタップ
2. **[システム]** 既存タグの候補を表示
3. **[ユーザー]** タグを選択、または新しいタグ名を入力
4. **[システム]** 選択/入力されたタグをチップとして表示
5. **[ユーザー]** カードを保存
6. **[完了]** タグがカードに紐付けられる

### 既存カードへのタグ追加
1. **[ユーザー]** カード一覧でカードを長押し、または編集画面を開く
2. **[システム]** タグ編集UIを表示
3. **[ユーザー]** タグを追加/削除
4. **[完了]** タグの紐付けが更新される

## 入力/出力

| 種別 | 項目名 | 型 | 必須 | 説明 | 制約・バリデーション |
|-----|-------|---|-----|-----|-------------------|
| 入力 | cardId | string | Yes | 対象カードID | UUID形式 |
| 入力 | tagIds | string[] | No | 付与するタグID | 既存タグのID |
| 入力 | newTagNames | string[] | No | 新規作成するタグ名 | 最大30文字/タグ |
| 出力 | card | Card | - | 更新されたカード | タグ情報含む |

## ビジネスルール

| ID | ルール | 条件 | 結果 |
|----|-------|-----|-----|
| BR-F016-01 | 複数タグ | 1カードに対して | 複数タグの付与が可能 |
| BR-F016-02 | タグ上限 | 1カードあたり | 最大10タグまで |
| BR-F016-03 | タグ自動作成 | 新しいタグ名入力時 | タグを作成してから紐付け |
| BR-F016-04 | 重複防止 | 同名タグ入力時 | 既存タグを使用（新規作成しない） |
| BR-F016-05 | タグなし許可 | タグ未設定 | 問題なく保存可能 |

## データモデル

### CardTag（中間テーブル）
```typescript
type CardTag = {
  card_id: string   // カードID
  tag_id: string    // タグID
  created_at: string
}
```

## エラーケース

| ID | 条件 | エラーメッセージ | 対処 |
|----|-----|----------------|-----|
| E-F016-01 | タグ数上限超過 | タグは10個までです | タグを減らす |
| E-F016-02 | タグ名が長すぎる | タグ名は30文字以内にしてください | タグ名を短くする |
| E-F016-03 | 保存失敗 | タグの保存に失敗しました | リトライ |

## 受け入れ条件（AC）

- [ ] AC-01: カード作成時にタグを付与できること
- [ ] AC-02: 既存カードにタグを追加・削除できること
- [ ] AC-03: 1つのカードに複数のタグを付与できること
- [ ] AC-04: 新しいタグ名を入力すると自動的にタグが作成されること
- [ ] AC-05: 同名のタグを入力した場合、既存タグが使用されること
- [ ] AC-06: タグが10個を超える場合、エラーが表示されること

## 画面要件

### タグ入力UI
- タグ入力フィールド（オートコンプリート付き）
- 既存タグのドロップダウンリスト
- 選択済みタグのチップ表示（×ボタンで削除可能）
- 新規タグ作成時の視覚的フィードバック
- タグ数カウンター（n/10）

### タグチップ
- タグ名の表示
- 色（ユーザー設定またはランダム割り当て）
- 削除ボタン（編集モード時）

## 技術仕様

### Supabase連携
```typescript
// タグ付け（トランザクション的に実行）
async function updateCardTags(cardId: string, tagIds: string[], newTagNames: string[]) {
  // 1. 新規タグを作成
  const createdTags = await createTagsIfNotExist(newTagNames)
  const allTagIds = [...tagIds, ...createdTags.map(t => t.id)]

  // 2. 既存の紐付けを削除
  await supabase.from('card_tags').delete().eq('card_id', cardId)

  // 3. 新しい紐付けを作成
  if (allTagIds.length > 0) {
    await supabase.from('card_tags').insert(
      allTagIds.map(tagId => ({ card_id: cardId, tag_id: tagId }))
    )
  }
}
```
