## 概要

ユーザーがカードの質問を見て答えを思い出し、トグルボタンで答えを表示して答え合わせを行う。これがSRS学習の基本フローとなる。

## ユーザーフロー

1. **[トリガー]** ユーザーが「学習開始」をタップ（F-020から）
2. **[システム]** カードを表示（質問のみ表示、答えは非表示）
3. **[ユーザー]** 答えを頭の中で思い出す
4. **[ユーザー]** 「答えを見る」ボタンをタップ
5. **[システム]** 同じカード内で答えをトグル表示（質問の下に答えが展開）
6. **[ユーザー]** 3段階で評価（F-022へ）
   - OK: 通常の復習
   - 覚えた: 復習完了
   - 覚え直し: 1日目からやり直し
7. **[システム]** 次のカードを表示
8. **[繰り返し]** 復習対象がなくなるまで 2-7 を繰り返す
9. **[完了]** 学習完了画面を表示

## 入力/出力

| 種別 | 項目名 | 型 | 必須 | 説明 | 制約・バリデーション |
|-----|-------|---|-----|-----|-------------------|
| 入力 | cardId | string | Yes | 学習中のカードID | UUID形式 |
| 入力 | showAnswer | boolean | Yes | 答え表示フラグ | トグルボタンで切替 |
| 出力 | card | Card | - | 表示中のカード | question, answer, tags |
| 出力 | progress | Progress | - | 学習進捗 | current, total, completed |

## ビジネスルール

| ID | ルール | 条件 | 結果 |
|----|-------|-----|-----|
| BR-F021-01 | 答え非表示で開始 | カード表示時 | 質問のみ表示、答えは非表示状態で開始 |
| BR-F021-02 | 答えトグル表示 | 「答えを見る」タップ | 答えをトグル表示（展開）、評価ボタンを表示 |
| BR-F021-03 | 答え再非表示 | 「答えを隠す」タップ | 答えを非表示に戻す（トグル操作） |
| BR-F021-04 | カード順序 | 学習セッション中 | 優先度順（overdue > new > due） |
| BR-F021-05 | セッション継続 | 評価完了後 | 自動的に次のカードを表示 |
| BR-F021-06 | 中断・再開 | セッション中断時 | 未評価カードは残り、再開可能 |

## 状態遷移

| 現在の状態 | イベント | 次の状態 | 備考 |
|-----------|---------|---------|-----|
| 答え非表示 | 「答えを見る」タップ | 答え表示 | 答えが展開、評価ボタン表示 |
| 答え表示 | 「答えを隠す」タップ | 答え非表示 | 答えを折りたたみ |
| 答え表示 | 評価ボタンタップ | 次カード（答え非表示） | スケジューリング更新 |
| 答え表示 | 最後のカード評価 | 学習完了画面 | セッション終了 |
| 任意の状態 | 「終了」タップ | ホーム画面 | 中断、未評価カードは残る |

## エラーケース

| ID | 条件 | エラーメッセージ | 対処 |
|----|-----|----------------|-----|
| E-F021-01 | カード取得失敗 | カードの取得に失敗しました | リトライ |
| E-F021-02 | セッションタイムアウト | セッションがタイムアウトしました | ホームに戻る |

## 受け入れ条件（AC）

- [ ] AC-01: カードに質問が表示され、答えは非表示であること
- [ ] AC-02: 「答えを見る」をタップすると答えがトグル表示（展開）されること
- [ ] AC-03: 「答えを隠す」をタップすると答えが非表示に戻ること
- [ ] AC-04: 答え表示後に3つの評価ボタン（OK/覚えた/覚え直し）が表示されること
- [ ] AC-05: 評価後、自動的に次のカードが表示されること
- [ ] AC-06: 学習進捗（現在/合計）が表示されること
- [ ] AC-07: 学習を中断しても、再開すると続きから始められること
- [ ] AC-08: 全カード完了後、学習完了画面が表示されること

### カード学習画面（答え表示）
```

│  [←] 学習中   3/15          │
│                             │
│  ┌─────────────────────┐   │

```

※ 答えは同じカード内で展開/折りたたみされる（カードをひっくり返すUIではない）

### 学習完了画面
- 学習したカード数
- 各評価の内訳（OK/覚えた/覚え直しの数）
- 学習時間
- 連続学習日数（ストリーク）
- 「ホームに戻る」ボタン
