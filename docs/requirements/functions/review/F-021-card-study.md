## 概要

ユーザーがカードの質問を見て答えを思い出し、トグルボタンで答えを表示して答え合わせを行う。これがSRS学習の基本フローとなる。

## ユーザーフロー

1. **[トリガー]** ユーザーが「学習開始」をタップ（F-020から）
2. **[システム]** カードを表示（質問のみ表示、答えは非表示）
3. **[ユーザー]** 答えを頭の中で思い出す
4. **[ユーザー]** 「答えを見る」ボタンをタップ
5. **[システム]** 同じカード内で答えをトグル表示（質問の下に答えが展開）
6. **[ユーザー]** 3段階で評価（F-022へ）
   - OK: 通常の復習
   - 覚えた: 復習完了
   - 覚え直し: 1日目からやり直し
7. **[システム]** 次のカードを表示
8. **[繰り返し]** 復習対象がなくなるまで 2-7 を繰り返す
9. **[完了]** 学習完了画面を表示

## 入力/出力

| 種別 | 項目名 | 型 | 必須 | 説明 | 制約・バリデーション |
|-----|-------|---|-----|-----|-------------------|
| 入力 | cardId | string | Yes | 学習中のカードID | UUID形式 |
| 入力 | showAnswer | boolean | Yes | 答え表示フラグ | トグルボタンで切替 |
| 出力 | card | Card | - | 表示中のカード | question, answer, tags |
| 出力 | progress | Progress | - | 学習進捗 | current, total, completed |

## ビジネスルール

| ID | ルール | 条件 | 結果 |
|----|-------|-----|-----|
| BR-F021-01 | 答え非表示で開始 | カード表示時 | 質問のみ表示、答えは非表示状態で開始 |
| BR-F021-02 | 答えトグル表示 | 「答えを見る」タップ | 答えをトグル表示（展開）、評価ボタンを表示 |
| BR-F021-03 | 答え再非表示 | 「答えを隠す」タップ | 答えを非表示に戻す（トグル操作） |
| BR-F021-04 | カード順序 | 学習セッション中 | 優先度順（overdue > new > due） |
| BR-F021-05 | セッション継続 | 評価完了後 | 自動的に次のカードを表示 |
| BR-F021-06 | 中断・再開 | セッション中断時 | 未評価カードは残り、再開可能 |

## 状態遷移

| 現在の状態 | イベント | 次の状態 | 備考 |
|-----------|---------|---------|-----|
| 答え非表示 | 「答えを見る」タップ | 答え表示 | 答えが展開、評価ボタン表示 |
| 答え表示 | 「答えを隠す」タップ | 答え非表示 | 答えを折りたたみ |
| 答え表示 | 評価ボタンタップ | 次カード（答え非表示） | スケジューリング更新 |
| 答え表示 | 最後のカード評価 | 学習完了画面 | セッション終了 |
| 任意の状態 | 「終了」タップ | ホーム画面 | 中断、未評価カードは残る |

## エラーケース

| ID | 条件 | エラーメッセージ | 対処 |
|----|-----|----------------|-----|
| E-F021-01 | カード取得失敗 | カードの取得に失敗しました | リトライ |
| E-F021-02 | セッションタイムアウト | セッションがタイムアウトしました | ホームに戻る |

## 受け入れ条件（AC）

- [ ] AC-01: カードに質問が表示され、答えは非表示であること
- [ ] AC-02: 「答えを見る」をタップすると答えがトグル表示（展開）されること
- [ ] AC-03: 「答えを隠す」をタップすると答えが非表示に戻ること
- [ ] AC-04: 答え表示後に3つの評価ボタン（OK/覚えた/覚え直し）が表示されること
- [ ] AC-05: 評価後、自動的に次のカードが表示されること
- [ ] AC-06: 学習進捗（現在/合計）が表示されること
- [ ] AC-07: 学習を中断しても、再開すると続きから始められること
- [ ] AC-08: 全カード完了後、学習完了画面が表示されること

## 画面要件

### ホーム画面のカードリスト形式（main.html）

v1モックでは、カード学習は専用画面ではなく**ホーム画面上のカードリスト形式**で行う。

#### コンポーネント構成

| コンポーネント | 役割 | 親コンポーネント |
|--------------|------|----------------|
| HomePage | ページルート | - |
| QuickInputForm | クイック入力エリア | HomePage |
| CardTabs | タブ切り替え（未学習/復習中/完了） | HomePage |
| CardList | カードリスト表示 | HomePage |
| StudyCard | 個別カード | CardList |
| CardTopBar | カード上部（評価ボタン+タグ） | StudyCard |
| RatingButtons | 3段階評価ボタン | CardTopBar |
| TagList | タグ表示 | CardTopBar |

#### タブ構成

| タブ | data-tab | 表示内容 |
|-----|---------|---------|
| 未学習 | tab-new | state='new' のカード |
| 復習中 | tab-review | state='learning' のカード |
| 完了 | tab-completed | state='completed' のカード |

各タブにはカード件数バッジを表示（例: `未学習 5`）

#### カード構造（StudyCard）

```
┌─────────────────────────────────────────┐
│ [CardTopBar]                            │
│ ┌─────────────────────────────────────┐ │
│ │ [OK]    [覚えた]   [もう一度]        │ │  <- RatingButtons（常時表示）
│ │ 3日後    完了      1日後            │ │
│ └─────────────────────────────────────┘ │
│ [タグ: 地理] [タグ: 日本]              │ │  <- TagList
├─────────────────────────────────────────┤
│ 日本の首都は？                    [編集] │  <- question + editボタン
├─────────────────────────────────────────┤
│         [ 答えを見る ]                  │  <- トグルボタン
├─────────────────────────────────────────┤
│ 東京                                    │  <- answer（トグル時のみ表示）
└─────────────────────────────────────────┘
```

#### 評価ボタン（RatingButtons）

| ボタン | class | ラベル | プレビュー |
|-------|-------|--------|----------|
| OK | rating-btn--ok | OK | 次の間隔（例: 3日後） |
| 覚えた | rating-btn--learned | 覚えた | 完了 |
| もう一度 | rating-btn--again | もう一度 | 1日後 |

**重要**: 評価ボタンは答え表示前から常時表示される（カード上部に配置）

#### 状態バリエーション

| 状態 | data-state/data-show-if | 表示 |
|-----|------------------------|------|
| 答え非表示 | !isOpen | トグルボタン「答えを見る」、answerエリア非表示 |
| 答え表示 | isOpen | トグルボタン「答えを隠す」、answerエリア表示 |
| 完了カード | isCompleted | RatingButtons非表示（タグのみ表示） |
| リスト空 | cards.length === 0 | 空状態メッセージ（Inboxアイコン + 「カードがありません」） |

#### ユーザーアクション

| 要素 | data-action | 説明 |
|-----|------------|------|
| 評価ボタン | onClick:onRate('ok'/'learned'/'again') | カード評価を送信 |
| 編集ボタン | onClick:onEdit | カード編集画面へ遷移 |
| トグルボタン | onClick:onToggle | 答えの表示/非表示切り替え |
| タブ | onClick:setActiveTab | タブ切り替え |

### 学習完了画面
- 学習したカード数
- 各評価の内訳（OK/覚えた/もう一度の数）
- 学習時間
- 連続学習日数（ストリーク）
- 「ホームに戻る」ボタン

## 技術仕様

### 学習セッション管理
```typescript
type StudySession = {
  cards: Card[]           // 学習対象カード
  currentIndex: number    // 現在のインデックス
  startedAt: Date         // 開始時刻
  results: StudyResult[]  // 評価結果

type StudyResult = {
  cardId: string
  rating: Rating          // 'ok' | 'learned' | 'again'
  reviewedAt: Date
  timeSpent: number       // ミリ秒
}
```
