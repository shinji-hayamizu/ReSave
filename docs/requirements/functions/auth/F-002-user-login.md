# F-002: ログイン

## 概要

登録済みユーザーがメールアドレスとパスワードでログインし、学習データにアクセスできるようにする。Supabase Authによるセッション管理。

## ユーザーフロー

1. **[トリガー]** ユーザーがログイン画面にアクセス
2. **[システム]** ログインフォームを表示
3. **[ユーザー]** メールアドレスとパスワードを入力
4. **[ユーザー]** 「ログイン」ボタンをタップ
5. **[システム]** 認証情報を検証
6. **[システム]** セッションを確立（JWTトークン発行）
7. **[完了]** ホーム画面へ遷移

## 入力/出力

| 種別 | 項目名 | 型 | 必須 | 説明 | 制約・バリデーション |
|-----|-------|---|-----|-----|-------------------|
| 入力 | email | string | Yes | メールアドレス | RFC 5322準拠 |
| 入力 | password | string | Yes | パスワード | 空文字不可 |
| 出力 | user | User | - | ユーザー情報 | id, email, created_at |
| 出力 | session | Session | - | 認証セッション | access_token, refresh_token, expires_at |

## ビジネスルール

| ID | ルール | 条件 | 結果 |
|----|-------|-----|-----|
| BR-F002-01 | 認証失敗 | メールアドレスまたはパスワードが不正 | 汎用エラーメッセージを表示 |
| BR-F002-02 | メール未確認 | メールアドレス確認が完了していない | ログイン不可、確認メール再送を促す |
| BR-F002-03 | セッション管理 | ログイン成功時 | JWTトークンをSecure Cookieに保存 |
| BR-F002-04 | 自動ログイン | 有効なセッションが存在 | ログイン画面をスキップしてホームへ |

## エラーケース

| ID | 条件 | エラーメッセージ | 対処 |
|----|-----|----------------|-----|
| E-F002-01 | 認証情報不正 | メールアドレスまたはパスワードが正しくありません | フォーム再入力 |
| E-F002-02 | メール未確認 | メールアドレスの確認が完了していません | 確認メール再送リンクを表示 |
| E-F002-03 | アカウントロック | アカウントがロックされています | サポートへ問い合わせ |
| E-F002-04 | サーバーエラー | ログインに失敗しました。再度お試しください | リトライ |

## 受け入れ条件（AC）

- [ ] AC-01: 正しい認証情報でログインすると、ホーム画面に遷移すること
- [ ] AC-02: 誤った認証情報の場合、汎用エラーメッセージが表示されること
- [ ] AC-03: メール未確認の場合、確認メール再送リンクが表示されること
- [ ] AC-04: ログイン後、ブラウザを閉じて再度開いてもログイン状態が維持されること
- [ ] AC-05: ログアウト操作でセッションが破棄されること

## 画面要件

### ログインフォーム
- メールアドレス入力フィールド
- パスワード入力フィールド（表示/非表示トグル付き）
- 「ログイン」ボタン
- 「パスワードを忘れた方」リンク（F-003へ）
- 「新規登録はこちら」リンク（F-001へ）
- Googleログインボタン（v1.1で追加予定）

### エラー表示
- フォーム上部にエラーメッセージを表示
- 入力フィールドのハイライト（エラー時）

## 技術仕様

### Supabase Auth連携
```typescript
// ログイン処理
const { data, error } = await supabase.auth.signInWithPassword({
  email,
  password
})

// セッション確認
const { data: { session } } = await supabase.auth.getSession()

// ログアウト
await supabase.auth.signOut()
```

### セッション管理
- access_token: 1時間有効
- refresh_token: 自動更新
- Secure, HttpOnly, SameSite=Lax Cookie
