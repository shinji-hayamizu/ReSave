# 画面遷移図

> 関連: [ビジネス要件](../requirements/business-requirements.md) | [機能一覧](../requirements/functions/_index.md)

## 設計方針

**ホーム画面ですべて完結する**

- ホーム画面内のタブで「未学習」「復習中」「完了」を切り替え
- 各タブにカードが表示され、その場で学習・評価が完了
- サイドバー（ドロワー）から設定・ユーザー詳細等に遷移
- カード作成は「クイック入力」と「詳細入力画面」の2パターン

## ホーム画面の構造

```mermaid
flowchart LR
    subgraph HomeScreen[ホーム画面]
        direction TB

        Header[ヘッダー<br/>ハンバーガーメニュー + 入力ボタン]

        subgraph Tabs[タブ切り替え]
            Tab1[未学習]
            Tab2[復習中]
            Tab3[完了]
        end

        subgraph Content[カード表示エリア]
            CardList[カード一覧<br/>タップで学習]
        end

        subgraph QuickInput[クイック入力エリア]
            QuickForm[テキスト + 隠しテキスト<br/>簡易保存]
        end
    end

    Header --> Tabs
    Tabs --> Content
    Header --> QuickInput
```

### ホーム画面のタブ

| タブ | 表示内容 | 操作 |
|-----|---------|------|
| 未学習 | まだ一度も学習していないカード | タップで初回学習 |
| 復習中 | 今日復習が必要なカード | タップで復習・評価 |
| 完了 | 「覚えた」評価で完了したカード | 閲覧 + 「覚え直し」ボタンで復習再開可能 |

### カード操作フロー（ホーム画面内）

```mermaid
flowchart TD
    ShowCard[カード表示<br/>評価ボタン常時表示] -->|答えを見るボタン| ShowAnswer[裏面表示]

    ShowCard --> Evaluate{自己評価}
    ShowAnswer --> Evaluate

    Evaluate -->|もう一度| Reset[間隔リセット<br/>復習中タブに残る]
    Evaluate -->|OK| Next[次の間隔へ<br/>復習中タブに残る]
    Evaluate -->|覚えた| Complete[完了タブへ移動]

    Reset --> NextCard[次のカードへ]
    Next --> NextCard
    Complete --> NextCard
```

**評価ボタン:**
- **OK**: 記憶OK、次の間隔へ進む（例: 3日後、7日後...）
- **覚えた**: 完全に記憶、完了タブへ移動
- **もう一度**: 忘れた、間隔リセット（1日後）

※ 完了タブのカードには「覚え直し」ボタンが表示され、復習サイクルを再開可能

## カード入力フロー

```mermaid
flowchart TD
    subgraph InputOptions[カード入力方法]
        Home[ホーム画面]

        Home -->|クイック入力| QuickSave[テキスト+隠しテキストのみ<br/>即座に保存]
        Home -->|詳細入力ボタン| InputScreen[カード入力画面へ遷移]

        QuickSave --> Home
        InputScreen --> DetailForm[詳細フォーム入力]
        DetailForm -->|保存| Home
        DetailForm -->|戻る| Home
    end
```

### クイック入力（ホーム画面内）

ホーム画面で最小限の情報だけを入力して即座に保存。

| 項目 | 必須 | 説明 |
|-----|-----|------|
| テキスト | 必須 | カードの表面（覚えたいこと） |
| 隠しテキスト | - | カードの裏面（答え） |

### 詳細入力画面

より詳細な情報を入力する専用画面。

| 項目 | 必須 | 説明 |
|-----|-----|------|
| テキスト | 必須 | カードの表面（覚えたいこと）※500文字制限 |
| 隠しテキスト | - | カードの裏面（答え） |
| タグ | - | 分類用タグ |
| ソース | - | 参照元URL |
| リピート | - | 間隔反復の設定 |

## 認証状態による分岐

```mermaid
stateDiagram-v2
    [*] --> CheckAuth: アプリ起動

    CheckAuth --> Unauthenticated: トークンなし/無効
    CheckAuth --> Authenticated: トークン有効

    Unauthenticated --> Login
    Login --> Authenticated: ログイン成功
    Login --> Register: 新規登録
    Register --> Authenticated: 登録完了

    Authenticated --> Home
    Home --> Unauthenticated: ログアウト
```
