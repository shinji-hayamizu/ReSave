Recommendation

  • Pattern 1 (Ghost) を推奨: 編集ボタンと統一感があり、hover時のみ危険色が現れるため視覚的にうるさくない
  • 削除ボタンは編集ボタンの右隣に配置し、アクション系をまとめる
  • ダイアログでは「学習履歴も削除される」という影響を明示して誤操作を防止
  • モバイルではダイアログのボタンを縦並びにしてタップしやすく
