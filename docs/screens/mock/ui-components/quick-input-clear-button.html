クリアボタン配置パターン v2

E: 入力欄内部にX（推奨）

Best

各入力欄の右端に小さなX。検索バーでおなじみのパターン

E-2: 空の時（Xは非表示）

入力がない時はスッキリ

F: 全体クリア1ボタン（右下に配置）

両方の入力欄を一括クリア

G: 編集ボタンがクリアに変化

入力がある時だけ編集ボタンがXに変わる

入力がない時は編集アイコン、入力があるとXに切り替わる

H: 丸いXアイコン（入力欄内部）

iOS風の丸いクリアボタン

I: 消しゴムアイコン（全体クリア）

消しゴムアイコンで直感的

J: ゴミ箱アイコン

削除のイメージが強すぎるかも

おすすめ順位

  1. 1. E（入力欄内部にX） - 最もスタンダード。Google/iOSでおなじみ
  2. 2. H（丸いX） - iOS風で視認性が高い
  3. 3. F or I（全体クリア1ボタン） - シンプル。ボタン1つで両方消える
  4. 4. G（トグル式） - ボタン数が増えないがUXが複雑
