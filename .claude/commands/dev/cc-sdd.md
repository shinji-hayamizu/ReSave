  - Phase 1-6 を順次実行
  - 各 Phase 完了後、次の Phase に自動で進む

  - コード生成 Phase（4.1-6.1）は並列 subagent で効率化

  1. 基盤: 1.1
  2. 解析: 2.1, 2.2
  3. 検出・選択: 3.1, 3.2

  5. 統合: 7.1, 7.2, 8.1
  6. テスト: 9.1, 9.2, 9.3
