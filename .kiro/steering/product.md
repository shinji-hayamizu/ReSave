# Product Overview

ReSave is a flashcard memorization app based on the Ebbinghaus forgetting curve. It uses Spaced Repetition System (SRS) to optimize review timing, enabling long-term memory retention with minimal study time.

## Core Capabilities

1. **Spaced Repetition Learning**: Fixed interval scheduling (1, 3, 7, 14, 30, 180 days) based on user self-assessment
2. **Card & Tag Management**: Create, edit, delete flashcards with multi-tag categorization
3. **Cross-Platform Sync**: Web (PWA) and Mobile (Expo) with unified data via Supabase
4. **Progress Tracking**: Daily study counts, streak tracking, tag-based mastery visualization

## Target Use Cases

- Exam preparation (certifications, entrance exams)
- Language learning (vocabulary, phrases)
- Professional knowledge acquisition
- Lifelong learning

## Value Proposition

- **Simplicity**: Clean Japanese UI unlike feature-heavy alternatives (Anki, Quizlet)
- **Scientific Approach**: Forgetting curve-based review scheduling
- **Accessibility**: Web + Mobile with seamless sync
- **Low Barrier**: Beginner-friendly onboarding
